beta  
  
##用法举例
------------- 
待定 如果需要主动发送消息，类似irc频道 指定对方名称(例如 66456445(昵称) )即可  
命令：/qsearch keyword #搜索好友列表，便于主动发送消息  
命令：/qpass password  #获取加密后的密码，储存安全  
命令：/qstatus busy    #修改用户状态 tab 自动补全
